# odin-recipes

TOP-Foundations-HTML: Recipes Project

Create a recipes website using HTML only, no CSS or JS.
This project is to practice using what has been taught so far:

- HTML
- Git
- GitHub
- VS Code

HTML:

- boilerplate (VS Code !)
- title (<title>)
- heading (<h1>-<h6>)
- paragraph (<p>)
- link (<a>)
- image (<img>)
- list and list items (ordered <ol> and unordered <ul>, <li>)
- comment (<!-- comment -->)

Git:

- git clone
- git status
- git log and git shortlog
- git add
- git commit (use conventional commits)
- git push

GitHub:

- create new repository
- view repository
- copy SSH data for git clone command

VS Code:

- panes (explorer, editor, and terminal)
- add files and directories (folders)
- code editor
- status bar (bottom bar)
- README file
- LICENSE file
- git commit using VS Code-Terminal (bash)
