# odin-recipes

TOP-Foundations-HTML: Recipes Project

Create a recipes website using HTML only, no CSS or JS.
This project is to practice using what has been taught so far:

- HTML
- Git
- GitHub
- VS Code

HTML:

- boilerplate (VS Code !)
- title (<title>)
- heading (<h1>-<h6>)
- paragraph (<p>)
- link (<a>)
- image (<img>)
- list and list items (ordered <ol> and unordered <ul>, <li>)
- comment (<!-- comment -->)

Git:

- git clone
- git status
- git log and git shortlog
- git add
- git commit (use atomic conventional commits, use VS Code editor)
- git push

GitHub:

- create new repository, include README and LICENSE (MIT) options
- view repository
- copy SSH URL for git clone command
  note: HTTPS, SSH, and GitHub CLI can be used to clone repositories

VS Code:

- panes (explorer, editor, and terminal)
- add files and directories (folders)
- edit HTML in VS Code editor
- git commit from VS Code-Terminal (bash) invokes commit msg editor in VS Code
- status bar (bottom bar) - get familiar with all the info and icons displayed
- add unknown words to workspace/user settings (cSpell extension)
- customize user and workspace settings (settings.json)
  user settings: ~/.config/Code/User/settings.json
  workspace settings: <project-dir>/vscode/settings.json
  workspace settings take precedence.
  note: defaultSettings.json, reference for all available options, read-only.
- edit README.md file
- view LICENSE file - GitHub created standard MIT License, looks good, no edits
